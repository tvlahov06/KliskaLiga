# Kliska Liga -- Generator Liga, Grupa i Knockout Sustava

**Kliska Liga** je interaktivna web aplikacija za kreiranje nogometnih
turnira u više formata: - LIGA sustav (svatko sa svakim) - GRUPNA faza
(više grupa + tablice + poredak) - KNOCKOUT (jedna utakmica ili dvomeč)

Aplikacija omogućuje jednostavno upravljanje utakmicama, unos rezultata,
automatsko računanje tablica i napredovanje kroz knock-out faze. Također
sadrži PWA podršku za instalaciju kao mobilna aplikacija.

## Funkcionalnosti

### Liga sustav

-   Automatsko generiranje utakmica prema broju timova\
-   Jednomeč ili dvomeč\
-   Unos rezultata\
-   Automatski izračun tablice

### Grupna faza

-   Podjela timova u više grupa\
-   Prikaz utakmica po grupama\
-   Automatska tablica\
-   Prva kola i uzvrati

### Knockout sustav

-   Generiranje playoff stabla\
-   Dvomeč podrška\
-   Agregat, gol u gostima i tie-break\
-   Napredovanje kroz faze

### PWA podrška

-   Instalacija aplikacije\
-   Offline rad\
-   manifest.json + service worker

### Ostalo

-   Dark mode
-   TailwindCSS UI
-   LocalStorage spremanje podataka turnira
-   Mobilna ptimizacija
